# 🚖 Uber Trip Analysis Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides an interactive analysis of Uber trip booking data, helping stakeholders monitor booking performance, revenue trends, trip completion rates, customer satisfaction, and vehicle utilization.

The dashboard enables users to explore key business metrics and gain actionable insights into booking patterns, operational efficiency, and customer experience.

---

## 📊 Key Metrics

* **Total Bookings:** 149K
* **Completed Bookings:** 93K
* **Lost Bookings:** 57K
* **Total Revenue:** 51.85M
* **Total Distance Covered:** 3M

---

## 📈 Dashboard Features

### Booking Analysis

* Total booking count tracking
* Completed, cancelled, and incomplete ride analysis
* Monthly booking trends

### Revenue Insights

* Revenue distribution by vehicle type
* Revenue trend analysis
* High-performing vehicle categories identification

### Vehicle Performance

* Comparison of vehicle categories:

  * Auto
  * Bike
  * Go Mini
  * Go Sedan
  * Premier Sedan
  * Uber XL

### Location Analysis

* Top pickup locations
* Top drop locations
* Geographic performance insights

### Customer & Driver Ratings

* Average customer ratings
* Average driver ratings
* Service quality monitoring

---

## 🎯 Business Insights

* Auto rides generated the highest revenue among all vehicle categories.
* More than 60% of bookings were successfully completed.
* Customer ratings remained consistently above 4.4, indicating high customer satisfaction.
* Specific pickup and drop locations contributed significantly to overall trip volume.
